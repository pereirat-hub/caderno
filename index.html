Integração com Google Drive

Insira sua API Key do Google e o ID da pasta onde os arquivos serão salvos como JSON.

Acesse console.cloud.google.com → Ative a Drive API → Crie uma API Key. O ID da pasta está na URL: drive.google.com/drive/folders/[ID]
